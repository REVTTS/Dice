# @REVTTS/Dice

Dice is a package for rolling dice.

## Installation

Use [yarn](https://yarnpkg.com/) to install @REVTTS/Dice.

```bash
yarn install @REVTTS/Dice
```

## Usage

```javascript
import Dice from '@REVTTS/Dice';

const dice = new Dice();

// Basic die
const result = dice.roll('1d20');
```
